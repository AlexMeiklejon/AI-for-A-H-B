# AI-for-Arts-And-Humanities-B

This project explores how a historical textile archive can be transformed into a structured dataset and analysed using computational tools.

## Project Overview
The project focuses on a catalogue of bark cloth (tapa) samples collected during Captain Cook’s voyages. The aim was to convert this physical archive into a CSV dataset and explore patterns within the material.

## Repository Structure

- `ai_arts_portfolio.ipynb` – Main project notebook (START HERE)
- `data/fabrics.csv` – Structured dataset of textile samples
- `images/` – Photographs of the textile catalogue and samples

## How to View

1. Open the notebook: `ai_arts_portfolio.ipynb`
2. Run the cells in order to view:
   - The dataset
   - Analysis
   - Visual outputs

## Key Features

- Manual dataset construction from archival material
- Data analysis using Python (pandas)
- Exploration of cultural and historical context
